# AI_ML_Internship-Task-3
# Exploratory Data Analysis (EDA)

## Dataset
Netflix Movies and TV Shows Dataset

## Objective
To analyze data patterns, distributions, and relationships using visualization techniques.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Steps
- Distribution analysis using histograms
- Categorical analysis using count plots
- Outlier detection using box plots
- Correlation analysis using heatmap
- Feature importance identification

## Key Insights
- Majority content released after 2010
- Movies dominate Netflix library
- USA and India lead content production
- Movie duration mostly within standard range

## Outcome
Clear understanding of data behavior and feature importance for prediction.
